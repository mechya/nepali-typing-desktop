# Nepali Keyboard — Phonetic (Romanized) for Windows

**Type Nepali anywhere on Windows by spelling words the way they sound.**
You type `namaste` → you get `नमस्ते`. Works in any app — Word, Chrome, Notepad, chat, email — as a system-wide input method.

## ✨ Features
- **Phonetic Romanized → Devanagari** — write Nepali using the English keyboard, no layout to memorize.
- **Works system-wide** — types directly into any Windows application.
- **Live suggestions** — a candidate list helps you pick the right word.
- **On-screen key chart** — see the romanization mapping anytime.
- **Private by design** — runs fully on your computer. No keystrokes, usage data, or personal information are sent anywhere.
