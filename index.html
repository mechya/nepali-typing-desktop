Linux — Ubuntu & Fedora new

An IBus input method. Enable in Settings → Keyboard → Input Sources, switch with Super + Space. Free.

⬇ .deb · Ubuntu ⬇ .rpm · Fedora

Install on Linux: sudo apt install ./ibus-nepali_*.deb  ·  sudo dnf install ./ibus-nepali-*.rpm  ·  then ibus restart
